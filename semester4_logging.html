Production Monitoring

Observing system health 24/7 after go-live

Production Monitoring Stack

ToolWhat It MonitorsFree Tier?
UptimeRobotURL availability every 5 minutes; sends email/SMS if downYes (50 monitors)
SentryApplication errors, unhandled exceptions, performanceYes (5k events/month)
Render MetricsCPU, memory, request volume for deployed serviceYes (built-in)
Supabase DashboardDatabase query performance, connection count, storageYes (built-in)
Logtail / PapertrailLog aggregation and search beyond Render's 7-day retentionYes (limited)

Key Production Metrics to Watch Daily

  • Uptime percentage (target: 99%+).
  • Number of 5xx errors per day (target: 0).
  • p95 API response time (target: < 500ms).
  • Active user sessions (business metric).
  • Orders placed per day (business metric).
  • Payment success rate (target: > 98%).

Incident Response

What to do when production goes down

An incident is any unplanned interruption to service. Having a response procedure means the team responds calmly and systematically rather than panicking.

Incident Response Playbook

StepActionExample for Food Delivery
1. DetectUptimeRobot or Sentry alerts teamEmail: "Your monitor food-delivery.render.com is DOWN"
2. TriageDetermine scope: is the whole site down or just one feature?Visit site → 500 error → check Render logs → DB connection error
3. CommunicateInform stakeholders (team + faculty guide)WhatsApp: "Production down, investigating. ETA 30 min."
4. MitigateRestore service ASAP — even if imperfect (rollback, restart)Render dashboard → Manual restart → Service recovers
5. ResolveFix the root cause properlyDB connection pool exhausted → increase pool size in settings
6. Post-MortemDocument the incident and lessons learnedWrite 1-page incident report: what happened, why, how to prevent
Incident Report is a Project Asset

If a production incident occurs during Semester 4, write a post-mortem and include it in your final report. It demonstrates professional maturity — acknowledging that failures happen and showing you responded systematically.
