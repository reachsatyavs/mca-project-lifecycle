Simple Dashboard with UptimeRobot + Render Metrics

For a student project, you don't need a full Prometheus/Grafana stack (that's production-grade). Use:

  • UptimeRobot (free) — uptime monitoring + downtime alerts by email/SMS.
  • Render Dashboard — built-in CPU, memory, and request metrics for deployed services.
  • Sentry Performance — request duration histogram and slow transaction alerts.

Custom Health Check Endpoint

# views.py — Health check endpoint from django.db import connection from rest_framework.response import Response from rest_framework.views import APIView class HealthCheckView(APIView): authentication_classes = [] permission_classes = [] def get(self, request): # Check DB connectivity try: connection.ensure_connection() db_status = "ok" except Exception: db_status = "error" status = "healthy" if db_status == "ok" else "unhealthy" http_status = 200 if status == "healthy" else 503 return Response({ "status": status, "database": db_status, "version": "1.0.0" }, status=http_status) # urls.py path('health/', HealthCheckView.as_view(), name='health-check')

Point UptimeRobot to https://your-app.render.com/health/ — it checks every 5 minutes and alerts you if the response is not 200 OK.

Structured Logs

Making logs machine-readable and searchable

Structured logging means writing logs as JSON objects rather than plain text strings. This makes them searchable, filterable, and parseable by log aggregation tools.

Plain Text vs Structured Log

Plain Text (Bad)Structured JSON (Good)
Order placed by user 42 for restaurant 7, total: 520 {"event":"order_placed","user_id":42,"restaurant_id":7,"total":520,"order_id":"ORD-001","timestamp":"2024-03-15T14:22:10Z"}

With structured logs, you can query: "Show all orders placed by user 42 in the last hour" or "Show all failed orders for restaurant 7" — impossible with plain text logs.

Log Retention

Keep logs for at least 30 days. On Render, logs are only kept for 7 days on the free tier. For longer retention, forward logs to Papertrail or Logtail (both have free tiers).
