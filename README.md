# MCA Project Lifecycle — End-to-End Reference

A complete 4-semester MCA software project reference, using an **Online Food Delivery System** as the worked example throughout.

[MCA Project Lifecycle](https://reachsatyavs.github.io/mca-project-lifecycle/)
---

## What's inside

| Semester | Topics Covered |
|---|---|
| **Semester 1 — Foundation** | Ideation & Scope · Requirements (SRS/FRs/NFRs) · User Stories · Use Cases · Test Planning · System Design (ER, Class, Architecture) · SPMP |
| **Semester 2 — Build** | Sprint Planning · API Contract · Flow & Sequence Diagrams · UI Wireframes · Node.js/Express Coding · Write Test Cases |
| **Semester 3 — Deliver** | Execute Test Cases · Alpha/Beta Testing · E2E Tests (Playwright) · Performance Tests · Docker · CI/CD · Logging |
| **Semester 4 — Maintain** | Security Testing · Production Deployment · Documentation · Maintenance |

## Tech stack used in examples

- **Backend:** Node.js 20 + Express.js
- **ORM:** Prisma
- **Database:** PostgreSQL
- **Frontend:** React.js (Vite)
- **Auth:** JWT (jsonwebtoken + bcryptjs)
- **Tests:** Jest + Supertest + Playwright
- **CI/CD:** GitHub Actions
- **Hosting:** Render (backend) + Vercel (frontend)
